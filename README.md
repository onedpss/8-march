## 🌸 О проекте

Этот сайт создан с любовью, чтобы поздравить мою дорогую маму с Международным женским днём!

### ✨ Что внутри:
- 🎨 Красивый дизайн
- 💌 Тёплые пожелания
- 📸 Красивая фотография

## 🚀 Как запустить

1. Скопируйте репозиторий
2. Вставьте фото мамы в папке `Photos` в формате .jpg и задайте имя Mom, чтобы получилось вот так: Mom.jpg
3. Откройте файл `index.html` в браузере
